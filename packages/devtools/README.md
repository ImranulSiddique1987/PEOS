# @peos/devtools

Shared development tooling workspace for the PEOS monorepo.

## Purpose

This package provides the foundation for reusable development tooling shared across the PEOS workspace.

Developer utilities, generators, automation scripts, and repository maintenance tools will be introduced incrementally as dedicated implementation milestones are completed.

## Status

Active workspace package.

Development tooling will be added only after its architecture and requirements have been validated.
