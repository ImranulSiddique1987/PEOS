# @peos/documentation

Shared documentation workspace for the PEOS monorepo.

## Purpose

This package provides the foundation for documentation assets shared across the PEOS workspace.

It intentionally contains only the documentation workspace infrastructure. Project documentation, architecture guides, ADRs, API references, and the PEOS Constitution are introduced only when their respective roadmap milestones are reached.

## Status

Active workspace package.

Documentation content will be added incrementally as validated implementation milestones are completed.
