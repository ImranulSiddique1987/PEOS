# @peos/core

Core runtime library for the PEOS (Product Engineering Operating System) monorepo.

## Purpose

This package serves as the foundational runtime library for shared platform functionality.

Future responsibilities may include:

- Domain primitives
- Shared utilities
- Common runtime abstractions
- Cross-package reusable logic

## Scope

This package intentionally starts with minimal implementation and will evolve as the PEOS platform grows.
