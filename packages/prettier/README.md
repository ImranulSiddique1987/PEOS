# @peos/prettier-config

Shared Prettier configuration for the PEOS monorepo.

## Available Configurations

### Default

```js
module.exports = require("@peos/prettier-config");
```

### React

```js
module.exports = require("@peos/prettier-config/react");
```

### Node

```js
module.exports = require("@peos/prettier-config/node");
```

## Purpose

This package provides a single source of truth for formatting across all PEOS applications and packages.
